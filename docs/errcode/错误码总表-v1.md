# 错误码总表 · v1

> **权威来源**：`pkg/errcode/errcode.go`。本文件由 Step 12 收口时生成，后续每次新增/修改错误码须同步更新此文件。
>
> **编码规范（XYYZZZ 六位三段式）**：
> - **X**：HTTP 分类前缀（4 = 所有模块统一使用；HTTP 实际状态由 `AppError.HTTPStatus` 字段决定，与 X 无绑定关系）
> - **YY**：模块号（00=通用 / 10=用户 / 12=内容 / 40=关注 / 50=搜索 / 60=上传 / 70=审核 / 80=加密）
> - **ZZZ**：模块内序号（从 101 起，预留 100 作为未来子模块分隔符）

---

## 通用段（400xxx / 401xxx / 403xxx / 404xxx / 429xxx / 500xxx / 503xxx）

> 所有模块共享，任意 handler 均可返回。不属于单一业务模块，YY=00。

| 码值 | 常量名 | HTTP | 含义 | 引入步骤 |
|---|---|---|---|---|
| `0` | `Success`（常量非 var） | 200 | 成功 | Step 1 |
| `400001` | `ErrInvalidParams` | 400 | 参数校验失败 | Step 3 |
| `401001` | `ErrUnauthorized` | 401 | 未登录 / Authorization header 缺失 | Step 3 |
| `401002` | `ErrTokenExpired` | 401 | Token 已过期 | Step 3 |
| `401003` | `ErrTokenInvalid` | 401 | Token 无效 / 已被黑名单 | Step 3 |
| `403001` | `ErrForbidden` | 403 | 无操作权限 | Step 3 |
| `404001` | `ErrNotFound` | 404 | 资源不存在（兜底） | Step 3 |
| `429001` | `ErrTooManyReq` | 429 | 请求过于频繁（限流） | Step 3 |
| `500001` | `ErrServer` | 500 | 服务器内部错误 | Step 3 |
| `500002` | `ErrDatabase` | 500 | 数据库错误 | Step 3 |
| `500003` | `ErrCacheError` | 500 | Redis 错误（已降级） | Step 3 |
| `503001` | `ErrServiceUnavail` | 503 | 服务不可用 / 依赖故障 | Step 3 |

## 审核模块（470xxx，HTTP 500，仅内部使用）

> YY=70。仅用于结构化日志 tag 和 Prometheus 告警，**不返回给 HTTP 调用方**。
> AuditConsumer 异步运行，所有错误仅落日志和 audit_logs 表，帖子保持 pending 状态。

| 码值 | 常量名 | HTTP | 含义 | 引入步骤 |
|---|---|---|---|---|
| `470101` | `ErrAuditAPIFailed` | 500 | 内容安全 API 调用失败 | Step 10 |
| `470102` | `ErrAuditWriteFailed` | 500 | 审核结果写入失败（audit_logs 或 posts 表） | Step 10 |

## 加密模块（480xxx，HTTP 500，仅内部使用）

> YY=80。仅用于结构化日志 tag 和 Prometheus 告警，**不返回给 HTTP 调用方**。
> 手机号 AES-GCM 加解密为透明基础设施，失败由 service 层 fallback 处理。

| 码值 | 常量名 | HTTP | 含义 | 引入步骤 |
|---|---|---|---|---|
| `480101` | `ErrEncryptPhone` | 500 | 手机号加密失败 | Step 11 |
| `480102` | `ErrDecryptPhone` | 500 | 手机号解密失败（GetMyProfile 降级为空串） | Step 11 |
